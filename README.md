# calculadora-enzimas

calculadora simples para ajudar no calculo de atividade enzimática.

hoje ela cobre:

- CAT
- GST
- SOD

fiz esse projeto para deixar esses calculos mais rápidos e organizados, sem precisar ficar montando tudo na mão toda vez.

o app foi feito em python com `tkinter` e roda como interface desktop.

para abrir:

```bash
python gui_calculadora_completa.py
```
